<h1 align="center">OIerDb</h1>

> OIerDb 是中国信息学竞赛选手的一个数据库。<br /><small>OIerDb is a database for Chinese OI participants.</small>

[🚧 **正在进行中**] 此仓库是 OIerDb 的新一代 Monorepo 仓库，包含前端、后端和数据生成器等多个子项目。
